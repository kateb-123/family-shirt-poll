# Family photos

Drop one photo per person into this folder and it will appear in their circle
on the poll. Name the file after the person, lowercase, as a .jpg or .png:

- jefe.jpg
- gram.jpg
- bryant.jpg
- charli.jpg
- kate.jpg
- tim.jpg
- madison.jpg
- emma.jpg
- isaac.jpg
- ian.jpg
- eileen.jpg
- rory.jpg

Square-ish photos look best (they get cropped to a circle). Until a photo
exists, the circle shows the person's first initial.

After adding photos, commit and push (or ask Claude to) so they show up on
the live site.
